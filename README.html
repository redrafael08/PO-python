Splief

Benodigdheden

  • Programma dat pythoncode uit kan voeren
  • Pygame bibliotheek

Programma starten

Voer het bestand "Splief.py" uit.

Speluitleg

Het doel van splief is de tegenstander door het veld met tiles te laten vallen. Dit kan gedaan worden door projectielen te schieten en te laten ontploffen. Deze ontploffingen vernietigen tiles en duwen de spelers weg.

Multiplayer

Host

Klik op Multiplayer en dan op Create Server.

Client

Klik op Multiplayer en dan op Join Server. Dan het ip-adres dat op het scherm van de host te zien is intypen en dan op join klikken.

Het programma werkt alleen als de host en de client aan hetzelfde netwerk verbonden zijn.
